技术型产品经理 / B端产品经理

岗位职责：
1. 负责 B 端 SaaS 产品的规划与设计，洞察企业客户需求
2. 编写 PRD 文档，绘制原型图，与研发团队高效协作
3. 跟进产品研发进度，协调资源，把控产品质量
4. 收集用户反馈，迭代优化产品功能和体验
5. 分析竞品，制定产品路线图和差异化策略
6. 与销售、客户成功团队协作，推动产品商业化落地

任职要求：
1. 本科及以上学历，3年以上 B 端产品经验
2. 有技术背景优先（计算机专业或有开发经验）
3. 深刻理解 B 端业务逻辑和企业客户需求
4. 熟练使用原型工具（Axure/Figma/Sketch）
5. 能看懂技术文档，与研发团队无障碍沟通
6. 了解常见技术架构（前后端分离、微服务、API 设计）
7. 数据驱动思维，能通过数据分析指导产品决策
8. 优秀的跨团队协作和项目推动能力

产品领域（优先）：
- CRM/ERP/OA 等企业管理系统
- 数据分析/BI 平台
- 开发者工具/PaaS 平台
- AI/大模型应用产品
- DevOps/低代码平台

技能要求：
- 产品设计：用户调研、需求分析、功能设计、交互设计
- 工具使用：Axure/Figma、Xmind、TAPD/Jira、SQL（加分）
- 技术理解：API 设计、数据库表结构、前后端交互、系统集成
- 沟通协作：PRD 编写、跨部门协调、客户需求管理

加分项：
- 有开发背景，会写代码（任何语言）
- 懂一些 UI/UX 设计原则
- 有成功的 0-1 产品经验
- 有 AI 产品经验（大模型、Prompt 工程）
- 了解敏捷开发流程（Scrum/Kanban）
- 有技术类产品的 ToB 销售支持经验

典型工作场景：
- 与客户深度访谈，挖掘潜在需求并转化为产品功能
- 设计复杂的权限体系和工作流引擎
- 平衡标准化产品与客户定制化需求
- 制定产品 API 开放策略，支持第三方集成
- 设计数据看板，帮助客户做业务决策

薪资范围：22K - 40K
工作地点：北京/上海/深圳
