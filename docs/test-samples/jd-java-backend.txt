Java 后端工程师

岗位职责：
1. 负责公司核心业务系统的后端开发和维护
2. 参与系统架构设计，保证系统的高可用性、高并发、高性能
3. 编写高质量代码，进行单元测试和代码审查
4. 解决线上问题，优化系统性能
5. 与产品、前端团队协作，推动项目按时交付

任职要求：
1. 本科及以上学历，计算机相关专业

3. 熟悉 MySQL、Redis 等数据库，有数据库调优经验
4. 熟悉分布式系统架构，了解微服务、消息队列（Kafka/RabbitMQ）
5. 熟悉 Linux 操作系统，有 Docker、K8s 使用经验优先
6. 有高并发系统开发经验者优先
7. 良好的代码规范和文档编写能力
8. 具备良好的沟通能力和团队协作精神

技术栈要求：
- Java 8+
- Spring Boot 2.x/3.x
- Spring Cloud（Nacos、Gateway、Feign）
- MySQL 5.7+
- Redis
- RabbitMQ/Kafka
- Git
- Maven/Gradle

加分项：

- 有开源项目贡献经验
- 了解前端技术（Vue/React）
