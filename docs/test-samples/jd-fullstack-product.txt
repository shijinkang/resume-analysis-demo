全栈工程师 / 产品技术负责人

岗位职责：
1. 负责产品从 0 到 1 的技术实现，独立完成前后端开发
2. 参与产品需求讨论，提供技术可行性建议和解决方案
3. 快速迭代 MVP，验证产品假设，根据用户反馈调整技术方案
4. 搭建基础设施（数据库、部署、监控），保证系统稳定运行
5. 随着产品发展，逐步组建技术团队并制定技术架构

任职要求：
1. 本科及以上学历，3年以上开发经验
2. 前端：熟练掌握 React/Vue 其中之一，能独立完成 UI 开发
3. 后端：熟悉 Node.js/Python/Java/Go 任意一门语言的 Web 开发
4. 数据库：熟悉 MySQL/PostgreSQL，了解 Redis 等缓存技术
5. 有独立负责过完整产品的经验，能从 0 搭建系统
6. 具备产品思维，能理解用户需求并转化为技术实现
7. 学习能力强，能快速上手新技术栈
8. 有创业公司或小团队工作经验优先

技术栈（不限于）：
- 前端：React/Vue/Next.js/Nuxt.js
- 后端：Node.js/Python Flask/FastAPI/Spring Boot/Go Gin
- 数据库：MySQL/PostgreSQL/MongoDB/Redis
- 部署：Docker/Nginx/云服务（阿里云/腾讯云/AWS）
- 其他：Git、RESTful API、WebSocket

加分项：
- 有 0-1 产品经验，参与过产品从启动到上线全流程
- 懂一些 UI/UX 设计，能用 Figma 画原型
- 会移动端开发（React Native/Flutter）
- 有技术博客或开源项目
- 对 AI/数据分析等新技术有兴趣和实践

工作特点：
- 小团队，扁平化管理，快速决策
- 业务导向，注重实际效果而非技术完美
- 灵活的技术选型，鼓励尝试新工具
- 与产品、设计紧密协作

薪资范围：25K - 40K + 期权
工作地点：北京/深圳/远程
