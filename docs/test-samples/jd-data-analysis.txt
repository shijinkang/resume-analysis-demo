数据分析师 / 数据开发工程师

岗位职责：
1. 负责业务数据分析，为产品和运营决策提供数据支持
2. 设计和维护数据指标体系，搭建数据看板（Dashboard）
3. 进行数据清洗、加工、建模，构建数据仓库
4. 开发数据管道和 ETL 流程，保证数据及时性和准确性
5. 探索数据中的业务洞察，输出分析报告
6. 与业务团队协作，理解需求并翻译为数据分析任务

任职要求：
1. 本科及以上学历，统计学、数学、计算机等相关专业
2. 2年以上数据分析或数据开发经验
3. 精通 SQL，能编写复杂查询和性能优化
4. 熟悉 Python，掌握 Pandas、NumPy 等数据处理库
5. 熟悉至少一种数据可视化工具（Tableau/Power BI/Metabase/Superset）
6. 了解数据仓库概念（维度建模、星型模型、ODS/DWD/DWS/ADS）
7. 有较强的业务理解能力和逻辑思维能力
8. 良好的沟通能力，能将数据洞察转化为业务语言

技术栈要求：
- SQL（精通）：MySQL/PostgreSQL/Hive/ClickHouse
- Python：Pandas、NumPy、Matplotlib、Seaborn
- 数据可视化：Tableau/Power BI/Echarts/Superset
- ETL 工具：Airflow/Kettle/DataX（了解）
- 大数据（加分项）：Spark/Hive/Flink

加分项：
- 有电商/金融/互联网业务分析经验
- 懂一些机器学习（分类、聚类、预测模型）
- 会 R 语言或 Julia
- 有 A/B 测试设计和分析经验
- 了解用户行为分析（埋点、漏斗分析、留存分析）
- 会写前端代码，能自己做数据看板

典型工作场景：
- 分析用户留存率下降原因，定位问题环节
- 为运营活动做效果评估（ROI、转化率）
- 搭建实时大盘监控，及时发现数据异常
- 构建用户画像和 RFM 模型，支持精准营销

薪资范围：18K - 30K
工作地点：北京/上海/杭州
